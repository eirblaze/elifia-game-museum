　エルグナディア民主主義人民共和国とセレナディア共和国は、エルテムスの南半球、ジュラムクロス大陸に位置する。
　以前、それらはひとつの国だった。しかし、戦争がこの国を二分してしまったのである。
　セレナディアは、北の豊かな大地によって、世界有数の食糧生産国として栄えることができた。
　一方、南のエルグナディアは、これといった資源も無い不毛地帯が、国土の大半を占めていた。さらに、政府の厳しい圧制のなかで、人々は悲惨な生活を強いられていたのである。
　そのためエルグナディアでは、北のセレナディアへ亡命する住民が相次いでいる。エルグナディア政府も、これに対して厳しい取締りを行っているという。

　エルグナディアの郊外に住む少年フレイと、その妹のフレイアも、セレナディアへの亡命を計画していた。
　彼らは、飛行機を操縦できるため、空港などから飛行機を奪って逃亡することにした。このほうが、地上よりはよほど安全だからである。
